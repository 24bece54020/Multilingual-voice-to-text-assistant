Project Name: Multilingual Voice to Text Assistant

Description:
This project helps users fill all government forms using their voice.
Supports multiple languages: Hindi, English, and Gujarati.
Automatically generates PDF from the voice input.

How to Run:
Open index.html in a browser.
Speak your details to fill the form.
Click "Generate PDF" to save the form.
